Разработать класс Map, представляющий 2 массива объектов типа T1 и Т2 одинакового размера.
Один массив хранит уникальные ключи, второй - значения.

Определить методы:
 - добавления элемента в Map
 - получения массива ключей
 - индексатор, который по заданному ключу устанавливает новое значение. Если ключа с заданным значением нет, добавляет новый элемент в Map
 - удаления элемента по ключу
 - преобразования в строку

Проверить работу этого класса, используя в качестве типа классы Person, Accоunt или Polynom.
Для текстового файла распечатать все слова и количество их повторов в файле.
